1. What is CSS?

CSS stands for Cascading Style Sheets. It allows you to apply styles to web pages, including colors, fonts, and layouts.

2. How to Add CSS

There are three ways to add CSS to an HTML document:

  • Inline CSS: Added directly to the element using the style attribute.
    • <p style="color: red;">This is red text.</p>
  • Internal CSS: Added inside a <style> tag in the <head> section.
    • <style>
    p {
        color: blue;
    }
</style>
  • External CSS: Linked via an external file using the <link> tag.
    • <link rel="stylesheet" href="styles.css">

3. CSS Selectors

Selectors are used to target specific elements for styling:

  • Element Selector: Targets all instances of an element.
    • h1 { color: green; }
  • Class Selector: Targets elements with a specific class. Use . before the class name.
    • .my-class { font-size: 18px; }
  • ID Selector: Targets an element with a specific ID. Use # before the ID name.
    • #my-id { background-color: yellow; }

4. Text Styling

You can change the color, size, and style of text using the following properties:

p {
    color: red;
    font-size: 16px;
    font-family: Arial, sans-serif;
    text-align: center;
}
