HTML Basics

Welcome to the **Stoner Web Academy's** guide to HTML basics! This is your roadmap to becoming a web design pro. Below is everything you’ll learn, starting from the basics to advanced topics.

1. What is HTML?

HTML stands for HyperText Markup Language. It’s the foundation of all websites and is used to structure web content.

2. Basic HTML Document Structure

<!DOCTYPE html>
<html>
    <head>
        <title>Page Title</title>
    </head>
    <body>
        <h1>Hello, World!</h1>
    </body>
</html>

3. Common HTML Tags

  • <h1> to <h6>: Headings
  • <p>: Paragraphs
  • <a href="url">: Links
  • <img src="url" alt="description">: Images

4. Lists

Unordered List:

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

Ordered List:

<ol>
    <li>Step 1</li>
    <li>Step 2</li>
</ol>

5. Links

<a href="https://stonerwebacademy.com">Visit Stoner Web Academy</a>

6. Images

<img src="image-url.jpg" alt="Description">

7. Tables

<table>
    <tr>
        <th>Name</th>
        <th>Age</th>
    </tr>
    <tr>
        <td>John</td>
        <td>25</td>
    </tr>
</table>

8. Forms

<form>
    <label>Name:</label>
    <input type="text" name="name">
    <button type="submit">Submit</button>
</form>

9. Future Topics

CSS Basics:

  • Styling text, colors, and fonts
  • Layout techniques (flexbox and grid)
  • Responsive design

JavaScript Basics:

  • Adding interactivity to websites
  • Event handling (e.g., clicks, hover)
  • Basic programming concepts (variables, loops, functions)
