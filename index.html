Comprehensive HTML Basics

Welcome to Stoner Web Academy’s full guide to mastering HTML. This guide will walk you through all essential topics, from basics to advanced concepts.

1. What is HTML?

HTML (HyperText Markup Language) is used to structure content on the web. It is the backbone of all web pages, defining the structure of headings, paragraphs, images, links, lists, and more.

2. Basic HTML Document Structure

An HTML document has the following structure:

<!DOCTYPE html>
<html>
    <head>
        <title>Page Title</title>
    </head>
    <body>
        <h1>Welcome to HTML</h1>
        <p>This is a basic HTML page.</p>
    </body>
</html>

3. Common HTML Tags

Headings

HTML provides six levels of headings, from <h1> to <h6>:

<h1>Main Heading</h1>
<h2>Subheading</h2>
<h3>Third-level heading</h3>

Paragraphs

Use the <p> tag to create paragraphs:

<p>This is a paragraph of text.</p>

Bold and Italic Text

Use <strong> for bold text and <em> for italicized text:

<strong>Bold Text</strong>
<em>Italic Text</em>

4. Lists

Unordered List

An unordered list displays items with bullet points:

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

Ordered List

An ordered list displays items with numbers:

<ol>
    <li>Step 1</li>
    <li>Step 2</li>
</ol>

5. Links

Links are created using the <a> tag:

<a href="https://stonerwebacademy.com">Visit Stoner Web Academy</a>

6. Images

Use the <img> tag to display images:

<img src="image.jpg" alt="Image description">

7. Tables

Tables are created using the <table> tag:

<table>
    <tr>
        <th>Name</th>
        <th>Age</th>
    </tr>
    <tr>
        <td>John</td>
        <td>25</td>
    </tr>
</table>

8. Forms

Forms collect user input:

<form>
    <label>Name:</label>
    <input type="text" name="name">
    <button type="submit">Submit</button>
</form>
