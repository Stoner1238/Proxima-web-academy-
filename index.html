Comprehensive HTML Basics

Welcome to Stoner Web Academy’s full guide to mastering HTML...

1. What is HTML?

HTML (HyperText Markup Language) is used to structure content on the web. It is the backbone of all web pages, defining the structure of headings, paragraphs, images, links, lists, and more.

2. Basic HTML Document Structure

<!DOCTYPE html>
<html>
    <head>
        <title>Page Title</title>
    </head>
    <body>
        <h1>Welcome to HTML</h1>
        <p>This is a basic HTML page.</p>
    </body>
</html>
