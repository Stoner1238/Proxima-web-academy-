Introduction to HTML

HTML (HyperText Markup Language) is the standard language used to create web pages. It provides the structure of a webpage and allows developers to define elements like headings, paragraphs, images, links, tables, and more.

1. Basic Structure of an HTML Document

Every HTML document follows a basic structure:

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Page Title</title>
  </head>
  <body>
    <h1>Hello, world!</h1>
  </body>
</html>

2. HTML Elements

HTML consists of elements. Each element has a start tag, content, and an end tag:

<p>This is a paragraph.</p>

3. Common HTML Elements

  • Headings: Used to define headings from <h1> (largest) to <h6> (smallest).
    • <h1>This is a heading</h1>
  • Paragraphs: Define text blocks.
    • <p>This is a paragraph.</p>
  • Links: Used to navigate to other pages or websites.
    • <a href="https://example.com">Visit Example</a>
  • Images: Embed images on a page.
    • <img src="image.jpg" alt="Description of the image">
  • Lists: There are ordered (numbered) and unordered (bulleted) lists.
    • <ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>
  • Tables: Used to display data in rows and columns.
    • <table>
  <tr>
    <th>Header 1</th>
    <th>Header 2</th>
  </tr>
  <tr>
    <td>Data 1</td>
    <td>Data 2</td>
  </tr>
</table>

4. Forms and Input Elements

Forms allow user input and interaction:

<form action="submit.php" method="post">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name">
  <input type="submit" value="Submit">
</form>

5. Semantic HTML

Semantic HTML provides meaning to the structure of a document:

  • <header>: Defines the header section.
  • <nav>: Defines navigation links.
  • <main>: Defines the main content of the page.
  • <article>: Represents self-contained content.
  • <footer>: Defines the footer section.

6. HTML Attributes

Attributes provide additional information about an element. Common attributes include:

  • href: Specifies the link URL for <a> tags.
  • src: Specifies the source for images.
  • alt: Provides alternate text for images.
  • id: Assigns a unique identifier to an element.
  • class: Assigns a class name for styling.

7. HTML Comments

Comments are not displayed in the browser and are used for documentation:

<!-- This is a comment -->

8. HTML Forms

Forms collect user input and send it to a server for processing:

<form action="/submit" method="post">
  <label for="email">Email:</label>
  <input type="email" id="email" name="email">
  <input type="submit" value="Submit">
</form>

9. HTML Best Practices

  • Use semantic tags for better structure and accessibility.
  • Validate your HTML to ensure it follows the proper syntax.
  • Use descriptive alt text for images.
  • Keep your code clean and organized with indentation and comments.

10. Conclusion

HTML is the foundation of web development. By mastering its structure, elements, and best practices, you can create well-structured, accessible, and user-friendly web pages.
