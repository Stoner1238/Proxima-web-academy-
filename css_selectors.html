CSS Selectors

Welcome to Stoner Web Academyâ€™s guide to CSS selectors. CSS selectors are used to target specific HTML elements for styling.

1. Universal Selector

The universal selector * targets all elements on the page:

* { margin: 0; padding: 0; }

2. Type Selector

The type selector targets elements by their tag name:

p { color: blue; }

3. Class Selector

The class selector targets elements with a specific class:

.my-class { font-size: 20px; }

4. ID Selector

The ID selector targets an element with a specific ID:

#my-id { background-color: yellow; }

5. Grouping Selectors

You can group multiple selectors to apply the same styles:

h1, h2, h3 { font-family: 'Arial'; }

6. Descendant Selector

The descendant selector targets elements nested inside other elements:

div p { color: green; }
